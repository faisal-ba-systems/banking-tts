⚡ What This System Does
🌐

5 Languages

English, Bangla, Japanese, German, and Spanish — with natural neural voices via Microsoft Edge-TTS.

🎙

Neural Voices

Male and female voice options for every language, powered by state-of-the-art neural TTS synthesis.

⚙

Full Control

Adjust speech rate, pitch offset, and volume for every announcement with fine-grained control.

⚡

Instant MP3

Announcements are generated in real-time and served as streamable MP3 files via REST API.

🔌 API Reference

All endpoints follow a consistent JSON envelope. See Swagger UI for full schema.

GET /tts/health Application health check
GET /tts/api/v1/voices List all supported languages and voices
POST /tts/api/v1/announcements Generate a multilingual TTS announcement MP3
GET /tts/static/generated_audio/* Stream or download generated MP3 files
